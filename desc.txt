A mod focused on goods, building, production methods and variability expansion, improvement and balancing.

This mod contains new buildings, goods, production methods and other various tweaks. Among them:
- Machine parts
- Sugar Substitutes
- Useful whaling
- Viable isolationism
- Teamster offices (Infra before railroads)
- Ore filtering
- Improved and extended automation
- New technologies
- Migration improvements
- Changes to Authority from laws (e.g., policing gives authority)
- [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3097932322]LRoAI[/url] (coming soon)


[h1]Localization and Compatibility[/h1]

[b]Chinese Localization[/b] - [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3003180347]Industry Expanded 汉化[/url] (Community-made)
Mod is made for game version [b]v1.5.12[/b]
Can be used with [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2889925770]Morgenröte[/url] if using a [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3126864902]Compatibility Patch[/url]
Should run normally with [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2935989855]VTM[/url] (IEX has to be placed [b]lower[/b] in loading order.)
Not compatible with [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3097932322]LRoAI[/url], but it will be integrated in the future.
Not recommended to use with mods, changing building or resource history.
Not recommended to use with mods changing PMs or buildings.
